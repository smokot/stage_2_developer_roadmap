# stage_2_developer_roadmap
Stage 2. Ethernet, HTTP/HTTPS, HTML, CSS


* Как работает Интернет
  - DNS, IP-адрес, TCP/IP, порт, сокет.
* Протокол HTTP и HTTPS
  - Структура HTTP-запроса: метод, путь, заголовки, тело.
  - Структура ответа: статус, заголовки, тело.
  - Основные заголовки: Host, User-Agent, Content-Type, Accept, Authorization, Set-Cookie.
  - Коды статусов: 2xx (успех), 3xx (перенаправление), 4xx (ошибка клиента), 5xx (ошибка сервера).
  - Отличие HTTP/1.1 от HTTP/2 и HTTP/3.
  - HTTPS: TLS-рукопожатие, сертификаты, цепочка доверия.
